# Le monde sur pause - Merchandise Website

This is a merchandise website for the EP "Le monde sur pause" featuring:

## Products

1. **CD Physique** - Mini-album de 6 titres
   - Price: 10€
   - Includes images of the CD recto and verso

2. **Livret de paroles et poèmes**
   - Price: 15€
   - Carnet relié à la main par Laoriane
   - Available in 3 colors: vert bleu, bordeaux, violet fleuris
   - Contains song lyrics and about twenty poems
   - Dimensions: A6 (10.5 x 15cm)

3. **Aquarelles**
   - Price: 3€
   - Postcard format, A6
   - 6 illustrations, one per song
   - Available with text in black or color
   - Artist: Lucie Bigaud
   - Graphic design: Eva Urbany

4. **Bougies fleuries**
   - Handmade candles
   - Made in recycled pots
   - Soy wax with dried flowers on top
   - Cotton wick
   - Each candle is unique
   - Small candle: 5€
   - Large candle: 8€

## Features

- Responsive design that works on all device sizes
- Interactive product selection options
- Image lightbox for detailed viewing with "Add to cart" button
- Persistent sticky shopping cart visible at all times
- Quantity badges on images showing current items in cart
- "Add to cart" functionality with visual feedback
- Clean, modern styling with smooth animations

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling
- `script.js` - JavaScript functionality
- `PHOTOS MERCH EP/` - Product images directory

## How to Use

Simply open `index.html` in a web browser to view the website. All functionality is self-contained and does not require a web server.

## Shopping Cart

The shopping cart is visible at all times in the top right corner. Click on the cart icon to view your items, quantities, and total. Each product image shows a badge with the current quantity in your cart.# laoriane-merch-EP-le-monde-sur-pause
